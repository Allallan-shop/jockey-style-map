# 騎手スタイル4象限マップ P0

過去レースデータから、騎手ごとの位置取り傾向とレース中の位置変化の目安を可視化した、閲覧用の静的HTMLレポートです。

GitHub Pagesでは `index.html` を入口として、主表示のmin100版、補助表示のmin50版、正方形レイアウト版を閲覧できます。

## 目的

過去騎乗データから、騎手ごとの「普段どの位置で運びやすいか」と「レース中にどの程度位置を動かすか」を比較するためのマップです。1人の騎手を1つの点で表し、騎乗スタイルの違いを相対的に確認します。

能力や騎乗の上手さを示すランキングではありません。

## マップの見方

- 上：前めの位置で運ぶ傾向
- 下：控えて後方から運ぶ傾向
- 右：レース中に位置を上げたり、自分から動いたりする傾向
- 左：位置を大きく変えず、溜めて待つ傾向

点が近い騎手ほど、この2軸上では位置取りと動かし方の傾向が近いことを示します。中心から離れるほど、その方向の特徴が相対的に強くなります。

### 4象限

- Q1：折り合いタイプ — 前め × 溜める・待つ
- Q2：前め積極タイプ — 前め × 動かす・押し上げる
- Q3：悠々自適タイプ — 控えめ × 溜める・待つ
- Q4：後方積極タイプ — 控えめ × 動かす・押し上げる

## 表示条件

- min100版：100騎乗以上の騎手を対象とする主表示。標本数を優先して確認する場合に推奨
- min50版：50騎乗以上の騎手を対象とする補助表示。対象は広がるが、min100版より偶然の影響を受けやすい

## 主な機能

- 騎手名の部分一致検索
- 騎乗数による表示フィルタ
- 直近5年に騎乗記録がある騎手だけを表示するフィルタ
- グラフ上の点へのhover／tapによる騎手情報表示
- min100主表示とmin50補助表示の切り替え

## 各ページ

- `index.html`：入口ページ
- `jockey_2axis_public_report_min100.html`：主表示版
- `jockey_2axis_public_report_min50.html`：探索・補助版
- `jockey_2axis_public_report_square.html`：正方形レイアウト版

## 注意事項

本レポートは、過去レースデータ上の位置取り・位置変化傾向を可視化した記述統計です。

騎手の能力、技術、判断、騎乗の良し悪しを直接評価するものではありません。馬の脚質、能力、厩舎方針、枠順、展開、距離、馬場、相手関係などは完全には分離できません。

使用データは前1走〜前5走ブロックから構成した過去騎乗データであり、全過去レースや全騎乗履歴を完全に網羅したものではありません。

特定の象限だけで、適性、勝率、回収率、馬券期待値を判断しないでください。

このレポートは学習・推論本線、production特徴量、馬券購入、自動購入には接続していません。

## 公開構成

このリポジトリには、GitHub Pagesで公開するために必要なHTML、CSS、JavaScript、JSON、CSV、画像のみを含めています。

内部分析ログ、生データに近い中間ファイル、ローカル環境のパス情報は含めていません。外部CDNに依存せず、静的ファイルだけで動作します。
