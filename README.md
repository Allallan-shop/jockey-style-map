# Jockey Style Map

騎手ごとの過去騎乗傾向を、2軸4象限で可視化する静的レポートです。

## 概要

Jockey Style Map は、過去レースデータ上の騎手ごとの位置取り傾向と位置変化の目安を、ブラウザ上で確認できるようにした公開用レポートです。

このページはP0試作版です。表示形式、データ形式、ラベル、説明文は今後変更される可能性があります。

## 4象限

| 象限 | ラベル     | 概要              |
| -- | ------- | --------------- |
| Q1 | 折り合いタイプ | 前め寄りで、位置変化は控えめ  |
| Q2 | 前め積極タイプ | 前め寄りで、押し上げ傾向も強い |
| Q3 | 悠々自適タイプ | 後方寄りで、位置変化は控えめ  |
| Q4 | 後方積極タイプ | 後方寄りで、押し上げ傾向が強い |

## 注意事項

このレポートは、過去レースデータ上の傾向を可視化するための参考資料です。

以下には使用していません。

* 競馬AI本線の学習
* 競馬AI本線の推論
* production特徴量
* 馬券購入判断
* 自動購入処理
* 投資助言
* 騎手能力の断定評価

表示されるスコアや象限は、あくまで過去データ上の位置取り・位置変化の傾向を見やすく整理したものです。レース条件、馬質、厩舎、距離、コース、時期、騎乗馬の脚質などによって実際の騎乗内容は変わります。

## 公開対象

この公開版には、表示に必要な最小限の集約済みファイルのみを含めます。

* HTML
* CSS
* JavaScript
* 表示用JSON
* 代表カードCSV
* preview画像

以下の内部ファイルは含めません。

* 生データに近いCSV
* dryrun fact
* 内部監査ログ
* source lineage
* ローカルパス情報
* score作成前の中間成果物
* 競馬AI本線・production・自動購入関連情報

## 位置づけ

このリポジトリは、騎手2軸4象限 report mart P0 の公開用静的サイトです。

競馬AI本線の開発リポジトリ、学習データ、推論パイプライン、馬券購入システムとは分離しています。
